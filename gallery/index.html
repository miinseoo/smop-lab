Gallery

Lab activities, conferences, and memorable moments

사진 추가 방법 — data/gallery.js 파일만 수정하면 됩니다

1. 이미지 파일을 images/gallery/ 폴더에 넣기
2. data/gallery.js를 열어 GALLERY 배열 맨 앞에 항목 추가:
   { title:"제목", date:"2025", category:"lab-life", image:"images/gallery/파일.jpg", size:null }
카테고리: lab-life / conferences / field-trips / celebrations
size: null(기본) / "wide"(가로 2칸) / "tall"(세로 2칸)
